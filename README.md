# Web Scraping

## Objective

The objective of this project is to collect data from a website by sending HTTP requests, parsing HTML content, extracting useful information, and organizing the extracted data for further analysis.

## Dataset

Dataset: Live Website Data

Source: Public Web Pages

## Tools and Technologies

* Python
* Requests
* BeautifulSoup (bs4)
* Pandas
* Jupyter Notebook
* VS Code

## Steps Performed

### 1. Import Required Libraries

Imported the necessary Python libraries for sending HTTP requests, parsing HTML, and handling extracted data.

### 2. Send HTTP Request

Connected to the target website using the Requests library and retrieved the webpage content.

### 3. Parse HTML Content

Parsed the HTML document using BeautifulSoup to navigate and locate required elements.

### 4. Data Extraction

Extracted relevant information such as text, headings, links, tables, or other required webpage elements.

### 5. Data Cleaning

Cleaned and structured the extracted data into a readable format.

### 6. Data Storage

Stored the extracted information in a Pandas DataFrame for further analysis or export.

* Ready for further analysis or visualization
